# 🧮 Calculator

A modern calculator built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations with a clean and responsive user interface.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🗑️ Clear display
- ⌫ Delete last character
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
